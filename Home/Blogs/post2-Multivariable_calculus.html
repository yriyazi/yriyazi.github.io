Continuity

We first need to revisit the continuity definition and derivatives. In the Real Analysis the \(\epsilon , \delta\) notation has been used.

From Wikipedia:

Suppose \(f:\mathbb{R} \rightarrow \mathbb{R}\) is a function defined on the real line, and there are two real numbers p and L. One would say that the limit of f, as x approaches p, is L and written \[\lim_{x \to p} f(x) = L\] or alternatively, say f(x) tends to L as x tends to p, and written: \[f(x) \to L \text{ as } x \to p,\] if the following property holds: for every real ε > 0, there exists a real δ > 0 such that for all real x, 0 < |x − p| < δ implies |f(x) − L| < ε. Symbolically, \[(\forall \varepsilon > 0 ) \, (\exists \delta > 0) \, (\forall x \in \mathbb{R}) \, (0 < |x - p| < \delta \implies |f(x) - L| < \varepsilon).\]
